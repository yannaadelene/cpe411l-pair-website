**PETMATCH**

PetMatch is a modern web-based directory designed to bridge the gap between future pet owners and their ideal companions by providing a centralized hub for breed discovery. The platform offers a clean, professional interface where users can explore the distinct personalities, physical requirements, and care needs of various dogs and cats. Built with a focus on accessibility and user experience, PetMatch simplifies the often overwhelming process of selecting a pet by delivering scannable, data-driven profiles in a visually engaging format.

**FEATURES**
- HomePage
- DogBreed Page
- CatBreed Page
- About Page

**How to Run Locally**
1. Clone or download this repository.
2. Open the folder.
3. Double-click index.html to view it in your browser.
